PROFIL PROFESSIONNEL

Depuis 2023, j'ai entamé une reconversion professionnelle motivante dans le domaine du développement web full stack avec une première étape via un DU à l'université de Strasbourg. Pour solidifier ces connaissances et me professionnaliser davantage, j'ai continué mon parcours avec la plateforme de e-learning Dyma.fr, spécialiser dans toutes les technologies du web. Puis pour aboutir à un diplôme RNCP je suis entrée dans l'école Studi où j'ai finaliser mes compétences en développement web full stack option Front-end. Je suis depuis passioné par la création d'application responsives avec des projets divers à mon nom. Je suis specialiser dans le langage python pour le back-end avec le framework Django. Mais je suis aussi alaise avec Symfony et PHP. En front je code avec React. Pour les bases de données, je suis formé au SQL avec MySQL et PostgreSQL mais je travaille aussi avec Mongodb pour le noSQL.
